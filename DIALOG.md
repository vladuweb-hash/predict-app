# Предскажи — История разработки

## Идея
Telegram Mini App «Предскажи» — игра-предсказание для набора аудитории.
Пользователи отвечают на вопросы-прогнозы (крипта, валюта, погода и т.д.), получают очки и соревнуются в рейтинге.

---

## Реализованные функции

### 1. Основная механика
- Вопросы с двумя вариантами ответа (A / B)
- +5 очков за участие, +20 за правильный ответ
- Категории вопросов: крипто, валюта, погода
- Таймфреймы: завтра, неделя, месяц
- Глобальный лидерборд
- Профиль с персональной статистикой

### 2. Автоматизация (scheduler.js)
- Ежедневная генерация вопросов из внешних API:
  - CoinGecko (BTC, ETH)
  - ExchangeRate API (USD/RUB, EUR/RUB)
  - OpenWeatherMap (погода, опционально)
- Автоматическая проверка и резолв вопросов по расписанию

### 3. Серия дней (Daily Streak)
- Ежедневный чекин с бонусными наградами
- Вехи: 3 дня (30 очков), 7 (100), 14 (250), 30 (500), 60 (1000), 100 (2000)
- Баннер серии с прогресс-баром
- Утренние напоминания через бота

### 4. Заморозки серии
- Восстановление потерянной серии
- Покупка за Telegram Stars (50⭐ = 1 шт, 100⭐ = 3 шт со скидкой)
- Покупка за очки (100 очков = 1 шт) — фолбэк
- Полный платёжный флоу: createInvoiceLink → openInvoice → pre_checkout → successful_payment

### 5. Комнаты (последняя фича)
- Создание приватных комнат (название + эмодзи, до 5 комнат)
- Вход по 8-символьному инвайт-коду
- Участники создают свои вопросы (до 5 в день)
- Автор / владелец комнаты выбирает правильный ответ
- Внутренний рейтинг среди друзей
- Шаринг кода через Telegram
- Deep link: ?room=КОД → автоматический вход

## Архитектура

### Файлы проекта
```
predict-app/
├── server.js          — Express API (авторизация, вопросы, прогнозы, комнаты, платежи)
├── bot.js             — Telegram бот (команды, уведомления, обработка платежей Stars)
├── database.js        — JSON-хранилище (users, questions, predictions, rooms)
├── scheduler.js       — Автогенерация и авторезолв вопросов
├── public/
│   └── index.html     — Фронтенд Mini App (SPA с табами)
├── data.json          — Файл базы данных
├── .env.example       — Шаблон переменных окружения
├── package.json       — Зависимости
└── README.md          — Документация
```

### Стек
- **Backend**: Node.js + Express
- **Database**: JSON-файл (fs)
- **Bot**: node-telegram-bot-api
- **Frontend**: Vanilla JS + Telegram Web App SDK
- **API**: CoinGecko, ExchangeRate, OpenWeatherMap
- **Платежи**: Telegram Stars (XTR)

### Переменные окружения (.env)
```
BOT_TOKEN=токен_бота
WEBAPP_URL=https://your-domain.com
PORT=3000
ADMIN_KEY=admin123
WEATHER_API_KEY=ключ_openweathermap (опционально)
```

---

## Монетизация
1. **Telegram Stars** — заморозки серии (основной канал)
2. **Реферальная система** — +50 очков за приведённого друга
3. **Комнаты** — виральный рост через инвайты

---

## Запуск
```bash
cd predict-app
npm install
# Создать .env из .env.example и заполнить BOT_TOKEN
npm start
```

Для Telegram Mini App нужен HTTPS (ngrok для тестов):
```bash
ngrok http 3000
```
